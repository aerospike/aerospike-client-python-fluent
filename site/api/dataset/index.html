    DataSet¶

    DataSet ¶

    DataSet(namespace, set_name)

    Pair a namespace and set name for building :class:~aerospike_async.Key values.

    Use :meth:of as the factory, then :meth:id for one key or :meth:ids for many. Pass keys to :meth:~aerospike_fluent.aio.session.Session.query, :meth:~aerospike_fluent.aio.session.Session.upsert, and other session APIs.

    Example

    users = DataSet.of("test", "users") k = users.id("user-123") ks = users.ids("a", "b", "c")

    See Also

    :class:~aerospike_fluent.aio.session.Session: Operations on keys.

    Create a DataSet instance.

    Parameters:

    Name Type Description Default
    namespace str

    The Aerospike namespace

    		 required
    set_name str

    The set name within the namespace

    		 required

    Raises:

    Type Description
    ValueError

    If namespace or set_name is empty

    Create an Aerospike key from a digest.

    Name Type Description Default
    digest Union[str, bytes]

    The digest as a hex string or bytes

    		 required

    Returns:

    Type Description
    Key

    A new Key instance created from the digest
    Example 
    users = DataSet.of("test", "users")
# From hex string

# From bytes

    id_for_object ¶

    id_for_object(obj)

    Create an Aerospike key from an object identifier. Supports String, Integer, Long, and byte array types.

    Parameters:

    Name Type Description Default
    obj object

    The object to use as the key identifier

    		 required

    Returns:

    Type Description
    Key

    A new Key instance

    Raises:

    Type Description
    TypeError

    If the object type is not supported
    Example 
    users = DataSet.of("test", "users")
key1 = users.id_for_object("user123")     # String key
key2 = users.id_for_object(123)           # Integer key
key3 = users.id_for_object(12345)         # Integer key
key4 = users.id_for_object(b"bytes")      # Bytes key

    ids ¶

    ids(*identifiers: str) -> List[Key]

    ids(*identifiers: int) -> List[Key]

    ids(*identifiers: bytes) -> List[Key]

    ids(identifiers: List[Union[str, int, bytes]]) -> List[Key]

    ids(*identifiers)

    Build many keys sharing this dataset's namespace and set.

    Call either with several positional identifiers (ids("a", "b", "c")) or with a single list (ids(["a", "b", "c"])). Mixed-type lists use :meth:id_for_object per element when the single-list form is used.

    Parameters:

    Name Type Description Default
    *identifiers Union[str, int, bytes]

    One or more identifiers, or exactly one list of identifiers.

    		 ()

    Returns:

    Type Description
    List[Key]

    A list of :class:~aerospike_async.Key instances in input order.

    Raises:

    Type Description
    TypeError

    If list elements are not supported key types.
    Example

    users = DataSet.of("test", "users") keys1 = users.ids("u1", "u2", "u3") keys2 = users.ids(["u1", "u2"]) keys3 = users.ids(1, 2, 3)

    See Also

    :meth:id: Single-key helper.
